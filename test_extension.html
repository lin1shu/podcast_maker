Test Page for Source URL Capture

This is a test page to verify that the Chrome extension can properly capture the current URL and send it as the source_url parameter.

To test:

  1. Select some text on this page
  2. Use the extension to process the text
  3. Check app_output.log to verify the source_url is being received

Here's some sample text you can select for testing: The quick brown fox jumps over the lazy dog. This simple sentence contains every letter of the English alphabet at least once.

Expected behavior:

  • When you select text and process it with the extension, the server logs should show a source_url value that matches this page's URL.
  • The MongoDB record for this processed text should include the source_url.
